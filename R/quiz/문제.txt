모집단과 표본은 무엇입니까?
모집단 : 연구자의 관심의 대상이 되는 모든 개체의 집합
표본 : 모집단에서 조사대상으로 채택된 일부

표본추출법을 아는대로 쓰시오.
표본을 추출하는 방법으로 대표적으로 단순랜덤추출 계통추출법 등이 있다

이산확률분포에 대하여 아는대로 쓰시오.
이산확률변수가 가지는 확률 분포
연속확률분포에 대하여 아는대로 쓰시오. 
확률 밀도 함수를 이용해 분포를 표현할 수 있는 경우

표본 분포가 무엇입니까?
확률표본의 확률변수의 분포

중심극한 정리가 무엇입니까? 
동일한 확률분포를 가진 독힙 확률 변수 n개의 평균의 분포는 n이 적당히 크다면 정규분포에 가까워진다는 정리

분산은 무엇입니까?
- 분포의 분산도 즉 관찰된 자료가 흩어져 있는 정도를 나타내는 개념

표준편차는 무엇입니까? 
- 분산을 제곱근 한 값으로 일반적인 분산도척도로 널리 쓰이는 개념

표준오차는 무엇입니까? 
표본평균의 표준편자

정규분포는 무엇입니까?
평균에 가까울수록 발생활 확률이 높고 평균에서 멀어질수록 발생할 확률이 적어지는 분포

표준정규분포는 무엇입니까? 
모든 정규분포에 공통적으로 사용할 수 있는 방법으로 표준편차를 사용한다

부트스트랩과 재표본추출의 차이점을 쓰시오. 
부트스트랩 : 복원 추출
재표본추출 : 비복원 추출 포함

배깅은 무엇입니까?
여러 부트스트랩 표본들로부터 얻은 예측값들을 모아서 결론을 만드는 것

귀무가설과 대립가설 에 대하여 쓰시오.
귀무가설 우연에 의한 결과라는 가설로 영가설이라고도 하며 H0로 표현된다
대립가설 : 귀무가설과 대조되는 가설로 증명하고자 하는 가설이다. H1으로 표현된다.
일원가설검정과 이원가설검정의 차이점은 무엇인가요?

유의수준과 유의확률에 대하여 쓰시오.
우연히 발생한 것인지 진짜 특별한 것인지 결정하는 기준으로 보통 0.05를 기준으로 한다.

1종오류와 2종오류에 대하여 쓰시오.
어떤 효과가 우연히 발생한 것인데 그것이 사실이라고 잘못 판단되는 경우
어떤 효과가 실제로 있는 것인데 그것이 우연히 발생한 것이라고 잘못 판단하는경우

t 검정은 무엇인가요? 
관찰된 효과가 정상적인 랜덤 변이 범위 내에 있는 지 여부를 판단하여 두 집단의 차이가 유의미한지 검정

ANOVA 검정은 무엇인가요? 
분산분석이라고도 하면 세개 이상의 여러 그룹간의 통계적 유의미한 차이를 검정하는 통계적 절차

카이제곱검정은 무엇을 평가하는 것인가요?
일반적으로 변수 간 독립성에 대한 귀무가설이 타당한지를 평가

단순선형회귀와 다중선형회귀의 차이점은? 
단순 :예측변수가 하나, 그래프가 직선으로 나타남
다중 : 예측변수가 여러개, 그래프가 직선 형태가 아님

선형회귀의 잔차/오차가 무엇인가요?  
실제 값과 예측 값의 차이

제곱근 평균제곱오차가 무엇인가요?
데이터 과학 관점에서 가장 중요한 성능 지표

결정계수가 무엇인가요? 
모델이 데이터에 잘 적합 하는 지 평가하는 것으로 0~1사이의 값

교차타당성 검사가 무엇인가요? 
학습할때는 사용하지 않는 데이터를 이용하여 모델의 선능을 평가하는 아이디어를 여러 개의 연속된 홀드 아웃 샘플로 확정한 것

stepAIC() 함수로 무엇을 할수 있나요? 
단계적 회귀 (예측 변수를 연속적으로 추가/삭제하여 AIC를 낮추는 모델을 찾는 것)

범주형 변수의 Onehot encoding이 무엇인가요? 
범주형 변수를 회귀 분석에 사용하기 위에 숫자로 변경 즉 수치화 하는 작업

다중공선성이 무엇인가요? 
독립변수들 간에 강한 상관관계가 나타나는 문제로 이를 제거해야 정상적인 회귀분석을 할 수 있음

교란변수가 무엇인가요? 
회귀방정식에 중요한 변수가 포함되지 못해서 생기는 누락의 문제

분류 예측 활용 사례에 대해 쓰시오.
